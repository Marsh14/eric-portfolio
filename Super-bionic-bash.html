Super Bionic Bash
Super Bionic Bash is a game that I am currently working on with my time at Limbitless Solutions. Limbitless focuses on making bionic arms for kids and they use video games to help them get used to the arms. Super Bionic Bash is a party style game and I am currently helping work on minigames for the game using Blueprints. I have worked on two minigames so far. The game is being developed in Unreal Engine and is intended to be used with a bionic arm controller but can be used with a normal controller also. I am currently working on it so I cannot provide pictures at the moment since it is still in development.
