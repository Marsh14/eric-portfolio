Campfire Cryptid
Campfire Cryptid was a horror survival game made in Unity for the Game Dev Knights Summer 2025 Game Jam. This was my first time working in Unity 6 so it was a great learning experience. I helped make the Shooting Eyes minigame in the game. I used Unity's new Input System to handle the movement of the crosshair and used Unity's UI to make the eyes spawn in random coordinates on the screen. I also helped implement the global data in the game to keep track of the night timer and the fire level. This game won the Community Favorite award in the jam.
