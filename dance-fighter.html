Dance Fighter
Dance Fighter is a fighting game developed in Unity made for the Knight Hacks Project Launch Event. Its an event where you work on a project the whole semester and present at the end in a demo. I was responsible for implementing the core health mechanics and using sliders, integrated end game logic by counting rounds, resetting the game after each round, and check to see if a player won a game by winning two rounds and making sure they cannot input moves after the round is over . I also implemented the timer in game and round countdown using Unitys Coroutine system. We ended up winning 3rd out of many projects at the Project Launch demo
